移动端开发框架Zepto.js入门

# 移动端与PC端框架的差异
- 开发中使用起来差异不大
- 差异主要在于框架内部的优化和处理
- 开发框架并不解决适配问题，多屏适配由Bootstrap这种框架解决
- 提供针对移动端事件的处理（触摸）、移动端设备的API（定位、发送短信）

# zepto和jQuery Mobile代表移动端框架的两个方向
- zepto： 轻量级，工具类库， 界面定制化程度高
- jQuery Mobile - 比较重，高度组件化，界面可定制化程度低
- zepto大量参考jquery API
- JQM = Jquery + UI组件， 包括CSS + JS

#zepto的好处
- 无缝接入和改造现有项目 （原来没考虑移动端的兼容，例如手势、触摸的处理）
- 更好的代码执行效率
- 学习成本低
- 轻量化，没有复杂的模式
